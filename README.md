# AeroNova

*AeroNova es un equipo formado para el concurso CanSat,*
*Con apoyo del Ies.Europa Móstoles, el equipo esta formado por:*
1. Christian Aceves

*Nuestra misión es medir el CO, CO2 y CH4 para ver la contaminación de la atmósfera.*

## Componentes
AeroNova utiliza los siguientes componentes:
1. 2 Arduino UNO R3
2. 1 BME280

## Librerías
AeroNova utiliza las siguientes librerias:
1. Adafruit BME280 Library

## Conexión
La conexión del circuito es la siguiente:

### TX
1. BME280
- 3Vo -> 3.3V
- GND -> GND
- SCK -> A5
- SDI -> A4
